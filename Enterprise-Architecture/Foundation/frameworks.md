# Enterprise Architecture Frameworks Overview

Enterprise Architecture frameworks help structure EA work and provide standard terminology.

## TOGAF (The Open Group Architecture Framework)
- Most widely adopted EA framework.
- Provides Architecture Development Method (ADM) with phases from Vision to Implementation.
- Includes guidelines for governance and standards.

## Zachman Framework
- Focuses on taxonomy (who, what, when, why, how, where).
- Useful to classify EA artifacts and views.

## Other Frameworks
- FEAF (Federal Enterprise Architecture Framework)
- Gartner EA Framework
- ISO/IEC 42010 — describes architectural description standards.

*Tip:* Choose a framework that fits your organization’s size, governance maturity, and goals.
