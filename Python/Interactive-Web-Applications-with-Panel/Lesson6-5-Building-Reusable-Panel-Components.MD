    param.Parameterized

---

# Why?

Because components become:

```text
Reactive
```

automatically.

---

# Example

```python
class StockCard(

    param.Parameterized
):

    symbol = param.String()

    price = param.Number()

    @param.depends(
        "price"
    )
    def view(self):

        return pn.indicators.Number(

            value=self.price
        )
```
