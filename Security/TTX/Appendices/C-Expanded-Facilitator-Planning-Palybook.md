# Appendix C — Expanded Facilitator Planning Playbook

> *Professional tabletop exercises are planned like projects, executed like operations, and reviewed like improvement programs.*

Most tabletop exercises last only a few hours.

Planning them properly often takes **four to eight weeks**.

The role of the facilitator extends far beyond moderating discussions. A facilitator is simultaneously a project manager, exercise designer, coordinator, communicator, moderator, and improvement leader.

This playbook provides a structured planning lifecycle that can be reused for every exercise, regardless of size or complexity.

---

# The Exercise Lifecycle

A mature tabletop exercise follows a repeatable lifecycle.

```text id="lifecycle-diagram"
Initiate
     │
     ▼
Plan
     │
     ▼
Design
     │
     ▼
Prepare
     │
     ▼
Rehearse
     │
     ▼
Execute
     │
     ▼
Evaluate
     │
     ▼
Improve
```

Each phase produces specific deliverables that become inputs to the next stage.

# Phase 1 — Initiation (T-60 to T-45 Days)

## Primary Objective

Establish the foundation for the exercise.

---

## Activities

☐ Identify executive sponsor.

☐ Define business objectives.

☐ Confirm exercise scope.

☐ Identify participating business units.

☐ Establish planning team.

☐ Define success criteria.

☐ Approve budget (if required).

☐ Reserve exercise date.

# Metrics for Success

Evaluate the planning process itself.

Examples include:

* Attendance rate
* Participant preparedness
* On-time delivery
* Inject timing adherence
* Completion of objectives
* Participant satisfaction
* Number of actionable findings
* Percentage of recommendations implemented

Tracking these metrics helps improve not only the exercise but also the planning process behind it.

---

# Continuous Improvement

After every exercise, ask the planning team:

* Which planning activities added the most value?
* Which activities created unnecessary effort?
* Which templates should be improved?
* Which checklists should be simplified?
* Which new scenarios should be developed?

Treat the planning process as a living system that evolves with each exercise.

---

# Facilitator Planning Principles

Successful facilitators think beyond the exercise itself.

They recognize that:

* Preparation determines confidence.
* Documentation enables consistency.
* Rehearsal reduces surprises.
* Observation creates evidence.
* Analysis drives improvement.
* Governance sustains progress.

An exceptional tabletop exercise is not the product of improvisation. It is the result of disciplined planning, thoughtful facilitation, structured evaluation, and a commitment to continuous organizational learning.
