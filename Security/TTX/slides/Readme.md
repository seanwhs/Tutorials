# Cybersecurity Tabletop Exercise (TTX) Masterclass

## Overview
This repository contains the comprehensive training materials for **"Conducting Effective Cybersecurity Tabletop Exercises."** This course is designed for security professionals, incident response teams, and leadership stakeholders to master the lifecycle of TTX—from initial scenario design to post-exercise remediation.

## Directory Structure
The curriculum is organized into modular slide decks to ensure ease of navigation and study:

*   `/slides/part-1-foundations`: Introduction, Value Proposition, and Planning Foundations.
*   `/slides/part-2-threat-modeling`: Participant Selection, Scenario Design, and MITRE ATT&CK Mapping.
*   `/slides/part-3-execution`: Facilitation Rules, Moderation, and The Inject Engine.
*   `/slides/part-4-post-mortem`: After-Action Reports (AAR), Remediation Roadmaps, and Pitfall Analysis.

## Curriculum Scope
This course covers the full spectrum of exercise orchestration:
1. **Strategic Value:** Demonstrating ROI to organizational leadership.
2. **Technical Preparation:** Defining scope and building realistic, risk-based threat models.
3. **Operational Execution:** Managing the "Inject Engine" to control pacing and cognitive load.
4. **Continuous Improvement:** Strategies for avoiding the "Blame Game" and driving actionable process change through AARs.

## Format Note
These slides have been generated as clean, minimalist text blocks to facilitate easy integration into presentation software (PowerPoint/Keynote/Google Slides). Each slide includes:
*   **Slide Title:** Key focus area.
*   **Bullet Points:** Concise teaching points.
*   **Speaker Notes:** Actionable facilitator guidance.

## Contributing / Customization
This is a living curriculum. As you adapt these exercises for specific industry sectors (e.g., Energy/Logistics or Finance), please ensure:
*   Scenario injects are updated to reflect the most current threat intelligence.
*   Remediation items are mapped to your internal organizational policies.
