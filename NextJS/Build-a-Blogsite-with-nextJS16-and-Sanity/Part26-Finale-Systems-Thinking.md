### Core Mental Models You Now Possess

1. **Everything is State**  
   Systems are defined by current state + allowed transitions.

2. **Everything is a Tree**  
   React, routing, layouts, data, failure, trust, observation — all hierarchical.

3. **Everything has Boundaries**  
   Clear separation of concerns, trust zones, and responsibilities.

4. **Everything is a Cache**  
   Human memory, browser cache, CDN, database cache — temporary copies for efficiency.

5. **Everything is a Tradeoff**  
   Speed vs correctness, simplicity vs power, security vs convenience.

6. **Everything is Information Flow**  
   From user → browser → server → database → CDN → back to user.

7. **Complexity is the Real Enemy**  
   The goal is not more features, but sustainable understanding.
