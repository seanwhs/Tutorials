# Appendix A35 — The Final Engineering Manifesto

This would be the final page of the entire repository.

Something like:

```text
Software engineering
was never about
writing code.

It was never about
frameworks.

It was never about
languages.

It was never about
clouds.

It was never about
AI.

Software engineering
has always been about:

Making decisions

under uncertainty

while managing
complexity

under constraints.

The tools change.

The principles remain.

Learn principles.

Practice judgment.

Expect failure.

Embrace uncertainty.

Continue learning.

And remember:

The objective
is not to
be correct.

The objective
is to become
less wrong
over time.
```
